# Dataset Information

The data folder is intentionally left empty in this repository due to GitHub's file size constraints and the massive scale of the dataset (1GB+). 

I will share the training dataset and relevant raw data directly with you through a secure cloud download link (e.g. Google Drive/AWS S3). Please download the data from that link and extract it into this `data/` folder before running the model training pipelines.

For any questions, reach out to me!
